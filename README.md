# termtetris
Tetris on the command line

## Getting started
Dependencies: a C++17 compatible Compiler, ncursesw, cmake and make

To start just execute: 
`mkdir build && cd build && cmake .. && make && ./termtetris`

## Controls
| Keys | Description |
| --- | --- |
| a, d | Move left |
| s | Move down twice as fast |
| r | Rotate |
| q | Quit the game |
